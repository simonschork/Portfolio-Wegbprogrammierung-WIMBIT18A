Deep Learning

Deep Learning ist ein Bereich des Machine Learning, indem künstliche neurale Netzwerke verwendet werden. um die gewünschten Ergebnisse zu erzielen. Ein solches Netz besteht aus mehreren Layern. Ein Layer wiederum besteht aus mehren Neuronen, welche die kleinste Struktur im Netzwerk darstellen. Jedes Neuron besteht im wesentlichen aus einer Funktion, die mit dem Input eine mathematische Berechnung durchführt. Input und Output eines Neurons könnten z.B. Vektoren sein. Der erste Layer eines Netzwerkes ist der Input Layer. Er erhält den Input vom Entwickler, meistens in Form eines Datensatzes, bestehend aus Beispielen. Dieser wird Train Set genannt. Der letzte Layer, der Output Layer, kann je nach Art des Netzwerks unterschiedlich aussehen. Im hier angewendeten Fall entspricht die Anzahl der Neuronen im Output Layer wahrscheinlich den möglichen Klassifizierungen, also sieben. Zwischen diesen beiden können theoretisch unendlich viele Hidden Layer liegen, praktisch wird die Anzahl durch die verfügbare Rechenleistung eingeschränkt. So wird im Idealfall das Ergebnis mit jedem Layer genauer.

Diese Überlegung und die dazugehörigen Algorithmen existieren schon sehr lange, das Konzept kam zum ersten Mal 1943 auf und orientiert sich tatsächlich am Aufbau eines menschlichen Gehirns. Doch damals war es unmöglich es umzusetzen. Heutzutage haben wir nicht nur genügend Rechenkraft, um künstliche neurale Netzwerke zu realisieren, die Menge an verfügbaren Daten ist drastisch gestiegen, was das Training solcher Strukturen erst möglich macht.

Im Gegensatz zu klassischen Algorithmen des maschinellen Lernens kann der Grenzwert, dem sich die Präzision des Algorithmus annähert, durch Hinzufügen von Hidden Layern gesteigert werden. Nachteil hierbei ist jedoch: Größere Netzwerke benötigen mehr Zeit für das Training.
